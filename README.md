# Mohr-Coulomb SIFEL Model

This repository now uses Tomas Koudelka's SIFEL-adapted UGN Mohr-Coulomb files as the active implementation:

- `mohrc_ugn.h`
- `mohrc_ugn.cpp`

Those files are kept as received from the mail attachment. The small compatibility files in this repository only let the SIFEL-facing source compile and run in this standalone test harness:

- `iotools.h`
- `global.h`
- `mechmat.h`
- `sifel_compat.cpp`

The compatibility layer is not a replacement for SIFEL. It is only enough to test the material-point overloads:

- `nlstresses(strain, eqstatev, stress, statev)`
- `stiffmat(strain, statev, stress, d)`
- `updateval(statev, eqstatev)`

## Interface

Voigt ordering:

- strain: `[eps_xx, eps_yy, gamma_xy, eps_zz]`
- stress: `[sig_xx, sig_yy, tau_xy, sig_zz]`

The current model is the 4-component 2D/plane-strain material-point implementation. It is not a 3D 6-component implementation.

## Per-Point Buffer

`eqstatev`:

| Range | Count | Meaning |
| --- | ---: | --- |
| `0..3` | 4 | converged plastic strain `epsp` |

`statev`:

| Range | Count | Meaning |
| --- | ---: | --- |
| `0..3` | 4 | current plastic strain `epsp` |
| `4..7` | 4 | previous plastic strain `epsp_prev` |
| `8` | 1 | `return_type` stored as `double` |

Total sizes:

- `eqstatev = 4`
- `statev = 9`

## Verification

Two standalone checks are included:

- `test_mohrc_ugn.cpp` runs finite-difference tangent checks for elastic, smooth, left-edge, right-edge, and apex returns.
- `compare_matlab_export.cpp` reads the snapshot generated by `matlab_export/export_loading_process_final.m` and compares plastic strain, previous-state copy, return type, stress, and reduced tangent against the MATLAB export.

Build and run:

```bash
make
./test_mohrc_ugn
./compare_matlab_export
```

To regenerate the MATLAB export first:

```bash
matlab -batch "cd('matlab_export'); export_loading_process_final; exit"
```

Expected results:

```text
MOHRC_UGN TEST STATUS: PASS
COMPARE STATUS: PASS
```
